This is a library set for numerical analysis, in this set you will find many methods:

 - **Regula Falsi**
 - **Newton-Raphson**
 - **Secant**
 - **Fixed Point**

##Polinomy simplification

 - **Horner**
 - **Gauss**
 - **Jacobi**
